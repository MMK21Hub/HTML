[reactrole|reactionrole|rr]
Reaction based roles is an intuitive way to offer your members the ability to add and remove roles to themselves. If you're just getting started using reaction roles, you're probably best off using !rr make
reactrole limit <msg> <limit>
Limits the number of roles you can pick up from one message at once.
reactrole link [messages...]
By linking two messages together, only one role from either message can be self-assigned. If you have 30 color roles for instance, linking the two messages together (since the limit is 20/message) allows a smooth, user-friendly experience when picking up roles. More than two messages can be linked together by using the same command a second time.
reactrole links
Shows the currently linked messages
reactrole lock <msg>
Locks a message and stops it handing out any roles.
Use "!help command" for more info on a command.
