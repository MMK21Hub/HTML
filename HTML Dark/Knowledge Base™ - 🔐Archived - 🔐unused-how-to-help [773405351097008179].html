--= Help us out! =-- Requirements 1. A GitHub account 2. Some basic knowledge on GitHub 3. Ask a Mod++ (or above) for the "GitHub bois" role Steps to help 1. Choose a FAQ to re-create, and react to the message with 2. Go to https://github.com/SheepCommander/KnowledgeBase/ and create a Pull Request 3. Make sure all the formatting of the resource is the same, "GitHub bois" allows you to use the message box so you can compare between yours and the original easily 4. Follow the file format:
faq <faq> (Don't worry too much about this part Reviewers will decide what to call it) raw-paste.txt example-pack.zip [Some resources don't need this]
5. Copy & Paste your recreation into raw-paste.txt 6. Add the following watermarks to the bottom
(Copy and Paste Extract from Knowledge Base: <https://discord.gg/xpNJdH9>) (Raw Paste & Downloadable Template: <https://github.com/SheepCommander/KnowledgeBase/tree/main/faq/[faq]>) NOTE: If there is no example-pack remove & Downloadable Template
7. Replace [faq] inthe link with what you put for the <faq> folder 8. Use base-datapack (https://github.com/SheepCommander/KnowledgeBase/tree/main/remaking-tools) as the base for example-pack.zip 9.
