Aeldrion 18-Nov-20 09:37 AM
Yeah, lists and arrays can only be filtered with a compound.
I'm not even sure a compound array or a string array is a thing in NBT.
If you want to know if a string is in a list, you can iterate through every item in the list: store the length of the list to a score, and as long as this score is positive, remove 1, copy the last string in the list to a separate storage tag and see if the value matches what you want, and remove the last string in the list.
Alternative you can outsmart it a little and try not to use a string list in the first place, make your strings keys in a compound instead of part of a list for example.
