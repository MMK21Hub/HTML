--= How to Detect Right Click on any COAS =-- (DP) (JE) (1.13+) (Using COAS)
Setup:
scoreboard objectives add RightClicked minecraft.used:minecraft.carrot_on_a_stick
Looping:
execute as @a[scores={RightClicked=1..}] run function namespace:right_click
right_click:
scoreboard players reset @s RightClicked <commands you want to run>

--= How to Detect Right Click on specific COAS =-- (DP) (JE) (1.13+) (Using COAS) (edited)
Give command:
give @s carrot_on_a_stick {Custom:1b}
Setup:
scoreboard objectives add RightClicked minecraft.used:minecraft.carrot_on_a_stick
Looping:
execute as @a[scores={RightClicked=1..}] run function namespace:detect
detect:
scoreboard players reset @s RightClicked execute if entity @s[nbt={SelectedItem:{tag:{Custom:1b}}}] run function namespace:right_click
right_click:
<commands you want to run>

--= Scoreboard Timer: =-- (DP) (JE) (1.13+)
+ You literally just count up by 1, detect a score, and add to the next value Here's an example: Setup:
scoreboard objectives add Timer dummy
Looping:
scoreboard players add $Ticks Timer 1 execute if score $Ticks Timer matches 20.. run function namespace:second
second:
scoreboard players set $Ticks Timer 0 scoreboard players add $Seconds Timer 1
