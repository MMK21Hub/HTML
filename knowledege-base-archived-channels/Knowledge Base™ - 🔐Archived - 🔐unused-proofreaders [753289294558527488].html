--= Commands tutorial! Part 1 =-- Let's start with the basics. We'll cover some basic solo commands in this one. To run a command in chat you need to put a / in front, followed by the command. In a command block it doesn't matter, with or without the / works. In a DataPack you have to leave out the / or it won't work. Now that we've established that, let's work on the actual command. The simplest and easiest to understand command is the /say command. The format (more properly the 'syntax') looks like this:
/say <text>
It will send a message in chat that will look something like this:
[YourName] Hello world!
You can put anything you want after the /say (don't forget the space) and it will repeat that in chat.
